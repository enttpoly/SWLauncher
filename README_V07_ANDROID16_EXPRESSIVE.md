# SW Launcher v0.7 Android 16 Expressive UI

Redesign inspirado em Material 3 Expressive e Android 16: home com cards expressivos, dock em ilha, busca central, app grid com placas tonais, configurações com cards de seção e ícone adaptativo com camada monochrome para themed icons.

Build mantém SDK 35 por compatibilidade com o GitHub Actions, mas a interface segue a linguagem visual do Android 16.
