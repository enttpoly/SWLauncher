# SW Launcher v0.6 Professional Foundation

Versão de base profissional: área de trabalho separada, gaveta de apps separada, dock funcional, configurações fullscreen e menu de app em bottom sheet.

## Inclui
- Workspace real com apps fixados
- Drawer separado com busca, categorias e A-Z
- Dock com botão central de gaveta
- Configurações em tela própria
- Menu ao segurar app em bottom sheet
- Paletas Neutral, Soft, Bright, Bold e AMOLED
- Ícone adaptativo inicial do APK
- Remoção de textos de protótipo

Observação: compila com SDK 35 porque o GitHub Actions atual não baixou android-37.
