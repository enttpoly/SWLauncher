# SW Launcher v0.4

Atualização de interface para uma direção Material 3 Expressive inspirada no Android 17.

Mudanças principais:
- Visual menos gamer neon e mais Material 3 Expressive.
- Paletas tonais: Pixel Violet, Android Blue, Mint Pulse, Rose Pop e AMOLED Ink.
- Header com card expressivo.
- Busca em formato pill.
- Abas com formato chip.
- Menu de app redesenhado.
- Configurações reorganizadas.
- Build preparado para Android 17 / API 37.

Observação: esta versão mantém a base Java/View para continuar compilando pelo GitHub Actions no mobile. Uma migração 100% Jetpack Compose + Material 3 oficial pode ficar para uma versão futura.
