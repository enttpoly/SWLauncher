# SW Launcher v0.15.1 — Official Adaptive Icon

Esta etapa aplica o ícone oficial **SW Aurora Icon** como ícone padrão do APK.

Implementado:

- Adaptive Icon Android em `mipmap-anydpi-v26`.
- Ícone legacy em `mipmap-mdpi`, `mipmap-hdpi`, `mipmap-xhdpi`, `mipmap-xxhdpi` e `mipmap-xxxhdpi`.
- Background Aurora roxo/azul para máscara adaptativa.
- Foreground com margem segura para evitar cortes em círculo, squircle, One UI e Android puro.
- `android:icon` e `android:roundIcon` atualizados no `AndroidManifest.xml`.

Identidade visual:

- Nome: SW Aurora Icon
- Estilo: Material 3, Aurora Glass, roxo/azul, brilho suave e borda segura.
