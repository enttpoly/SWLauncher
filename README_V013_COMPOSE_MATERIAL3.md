Migração da interface para Kotlin + Jetpack Compose + Material 3.

- tema Material 3 centralizado;
- Dynamic Color no Android 12+;

- bottom sheet de ações de app;
- configurações salvas;
- ícone adaptativo do APK;
- animações leves de transição.
