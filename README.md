# SW Launcher

Launcher Android criado e mantido pelo celular usando Termux + GitHub Actions.

## v0.2

- Tela inicial com apps fixados.
- Mover apps fixados para esquerda/direita.
- Editar nome do app dentro do launcher.
- Ajustar tamanho do ícone por app.
- Ajustar tamanho global dos ícones.
- Ocultar/mostrar apps.
- Filtros: Início, Apps, Recentes e Usados.
- Rolagem alfabética A-Z.
- Menu de configurações do launcher.
- Atalhos para telas oficiais de cache/dados do Android.
