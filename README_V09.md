# SW Launcher v0.9 Flexible Layout

Nesta versão:
- layout mais compacto e limpo;
- densidade visual Compacta/Confortável;
- estilo da área Focus/Clean/Creator/Minimal;
- dock Flutuante/Compacto/Sólido;
- animações Suaves/Reduzidas/Desligadas;
- A-Z menor e menos invasivo na gaveta;
- ícones internos por vetores, sem círculos com letras;
- home separada da gaveta;
- dock salvo;
- nomes personalizados salvos;
- apps ocultos salvos;
- onboarding inicial;
- paletas Pixel Porcelain, Android Soft, Android Bright, Android Bold e AMOLED Ink;
- ícone adaptativo do APK com camada monocromática.

Observação: cache/dados abre a tela oficial de informações do app, porque Android não permite limpar dados de outros apps sem privilégio de sistema/root.
