# SW Launcher v0.6

Redesign de base: área de trabalho + gaveta de apps, inspirado em launchers premium, com linguagem Material 3 Expressive simplificada em Java nativo.

## Principais mudanças
- Tela inicial separada da gaveta de apps
- Dock funcional com botão de gaveta
- Apps fixados na área de trabalho
- Gaveta com busca e categorias visuais
- Menu ao segurar app: abrir, fixar, dock, editar nome, ocultar, info e desinstalar
- Configurações em tela dedicada
- Paletas Neutral, Soft, Bright, Bold e AMOLED

Observação: o app compila com SDK 35 porque o GitHub Actions não encontrou android-37 no ambiente atual.
