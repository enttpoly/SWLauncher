# SW Launcher v0.12 — Real Workspace Setup

Esta versão move a SW Launcher para uma base de produto mais real:

- onboarding inspirado em launchers profissionais;
- tela para definir launcher padrão;
- ajustes de pesquisa, ícones e gaveta;
- tela de configurações mais completa;
- home Aurora Glass refinada;
- menu de app com ações reais;
- persistência local de tema, ícones, home, dock, ocultos e nomes editados;
- gestos básicos: deslizar para cima abre gaveta, deslizar para baixo abre personalização.

Próxima etapa recomendada: drag-and-drop real da área de trabalho + pastas.
