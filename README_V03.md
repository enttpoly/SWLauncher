# SW Launcher v0.3

Atualização visual gamer com SW Core UI.

Novidades principais:
- Tema Cyber Violet como padrão visual.
- Temas extras: ROG Red, Cyber Blue, Toxic Green, AMOLED Pure e Ice Neon.
- Header gamer com status Game Core, modo Turbo e bateria.
- Painel Turbo visual com Balanced, Performance, Turbo e Battery.
- Busca redesenhada.
- Abas e categorias com visual gamer.
- Categorias rápidas: Todos, Jogos, Social, Mídia, Ferramentas e Sistema.
- Menu ao segurar app em painel customizado.
- Configurações reorganizadas por área.
- Dock gamer.
- Estados vazios melhores.
- Barra A-Z mais integrada.

Observação:
Algumas funções profundas como limpar dados/cache diretamente, alterar CPU/GPU ou mudar ícone real do app dependem de permissões do Android, sistema, root ou APIs específicas. Nesta versão elas são feitas como atalhos seguros ou preparação para versões futuras.
