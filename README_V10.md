# SW Launcher v0.10 Aurora Glass Foundation

Direção visual baseada no conceito Aurora Glass:
- fundo em gradiente pastel roxo/azul/rosa;
- cards translúcidos tipo vidro fosco;
- widget visual de relógio/clima;
- busca em formato pill;
- dock translúcido;
- ícones com placas arredondadas;
- painel Personalizar com abas Temas, Ícones e Layout;
- estilos Aurora, Mist, Dusk, AMOLED e Bold;
- mantém persistência local de home, dock, ocultos, nomes, tema, layout e onboarding.

Observação: clima ainda é visual/simulado nesta etapa. Clima real, widgets reais, pastas avançadas, badges e shortcuts entram nas próximas fases.
